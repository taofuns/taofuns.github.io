How to use

  1. git clone https://github.com/JohnSundell/Publish.git
  2. cd Publish
  3. make (make command will install CLI on Mac, also can use brew install publish)
  4. publish new to init template
  5. publish deploy or publish run

Workflow: use Publish to design website use Obsidian to manager posts add metadata upload picture to PicGo * use Github page to deploy
